# Password_Generator_Tkinter_tool
A Tkinter  based password generator for creating strong and secure passwords.

## Project Description
The Password Generator Tool is a Python-based desktop application developed using Tkinter. It helps users generate strong, random, and secure passwords through a simple graphical user interface. This tool enhances password security by creating passwords that include a combination of letters, numbers, and special characters.

## Features
- Generate strong and secure passwords
- User-friendly graphical interface
- Random password generation
- Fast and efficient performance
- Easy to use

## Technologies Used
- Python
- Tkinter

## How to Run the Project
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

bash
main.py


5. The application window will open.
6. Click the generate button to create a secure password.
